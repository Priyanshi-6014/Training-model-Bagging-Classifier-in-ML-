Project Overview:
This project demonstrates the implementation of a Bagging Classifier using Decision Trees to improve model performance and reduce overfitting.
Here,I had train a decision tree classification model on telecom customer churn dataset and use the bagging ensemble method to improve the performance. 

Goal:
To compare performance:
Without Bagging
With Bagging

Resut:
Bagging improves model stability and accuracy by combining multiple decision trees.
